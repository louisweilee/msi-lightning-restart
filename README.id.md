\# Alat Restart MSI Lightning Service



Utilitas pemulihan ringan untuk sistem MSI GeForce RTX 5090 LIGHTNING Z.



Alat ini menjalankan ulang `MSI Lightning Service.exe` saat layar bawaan pada kartu grafis berhenti memperbarui data pemantauan perangkat keras secara real-time.

\## Ringkasan



Utilitas ini ditujukan untuk kondisi ketika layar terintegrasi pada kartu grafis berhenti memperbarui data pemantauan sistem secara real-time.



Skrip ini melakukan tindakan berikut:



\- Menghentikan `MSI Lightning Service`

\- Menjalankan ulang `MSI Lightning Service.exe`




\## Berkas yang disertakan



\- `restart\_msi\_lightning.ps1`

\- `run\_restart\_msi\_lightning.bat`



\## Cara penggunaan



1\. Simpan kedua berkas skrip di folder yang sama.

2\. Klik ganda `run\_restart\_msi\_lightning.bat`.



\## Jalur layanan



Skrip saat ini mengasumsikan jalur executable berikut:



`C:\\Program Files (x86)\\MSI\\PortalX\\MSI Lightning\\MSI Lightning Service.exe`



\## Catatan



\- Utilitas ini bukan alat resmi dari MSI.

\- Validasi hanya dilakukan pada sistem yang menggunakan jalur layanan seperti di atas.

\- Pengguna bertanggung jawab menilai kesesuaian dan menanggung risiko penggunaan.
